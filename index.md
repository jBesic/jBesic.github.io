## About Me
Hi 👋🏼 My name is Jasmin Besic. I started coding in 2015, and I like programming and helping others become programmers.

### About This Blog
This blog is my platform for making notes and helping others to learn something about coding from me. With each post, I try to delve deeper into the ever-expanding universe of web development. I strongly believe that inspiration is for amateurs. So every day I try to practice web development for couple hours before my regular job.

My aim is to write and share posts every week on some aspect of web application development including the topics of JavaScript, HTML, Cascading Style Sheets (CSS), PHP and SQL but not limited to these.
